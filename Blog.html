Question-1: What are the different ways to select an element in the DOM?

In the DOM, elements can be selected using various methods. The `getElementById("id")` selects an element by its unique ID, while `getElementsByClassName("class")` and `getElementsByTagName("tag")` return collections of elements based on class and tag names. The `querySelector("selector")` selects the first matching element, whereas `querySelectorAll("selector")` retrieves all matching elements as a NodeList.

What is the difference between innerHTML, innerText, and textContent ?

The `innerHTML` property sets or gets the HTML content inside an element, allowing it to render HTML tags. The `innerText` retrieves or modifies the visible text inside an element, ignoring hidden elements and preserving styling. The `textContent` returns or sets the full text content of an element, including hidden elements, without parsing HTML.

What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using **event bubbling**. Instead of adding listeners to multiple child elements, a single listener on the parent captures events as they propagate upward. This improves performance and is useful for dynamically added elements.

What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the **target element** and propagates **upward** through its parent elements in the DOM. It allows event handlers on parent elements to listen for events triggered on their child elements. This behavior is useful for **event delegation** and reducing multiple event listeners.
